# Публикация темы на MarketPlace 

## После долгого простоя

Если вы долго не опубликовали материал, вероятно ваш ключ доступа иначе называемый токен доступа устарел и требует обновления. При попытке выполнения команды обновления в терминале

```sh
vsce publish
```

…вы получите предупреждение об ограничении доступа и предложение создать новый токен. Перейдите по ссылке создайте токен. Ссылка ведёт на документацию, в которой указаны шаги по созданию ключа доступа – токена. После того как вы получите токен доступа вы выполните в терминале команду:

```sh
vsce login  <yourename>  
```
>`yourename` – обычно ваше имя на маркетплейс, которое также указывается в файле package.json

Далее вам будет предложено ввести токен доступа. Введите нажмите Return или Enter. Далее выполните команду публикации в терминале:

```sh
vsce publish
```
